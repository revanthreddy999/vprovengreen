# Vprove Corporate React (Green Theme)

React + Vite + Tailwind project tailored with a greener palette, updated team, and animated 3D cards.

## Dev
```bash
npm install
npm run dev
```
## Build
```bash
npm run build && npm run preview
```
## Deploy (Vercel)
- Dashboard → Add New → Project → **Upload** → drop this folder  
- or connect GitHub and **Import**.

Images are in `src/assets`. Update roles/photos as needed.
